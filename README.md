# Mousescreen
This project is a test to interface with a usb mouse and output it to a 640x480 screen via hdmi/dvi
The FPGA used is a tang primer 25k which uses a GW5A-LV25MG121NC1/I0
To use, connect mouse to usb port and connect HDMI PMOD to slot closest to usb c port
for the other 2 PMOD put LEDs there
